# StorageClass.info

StorageClass.info is a curated CSI driver list for Kubernetes. It helps teams compare CSI drivers by storage type, lifecycle support, and feature capabilities.

## What it includes
- Searchable CSI driver directory
- Filters for storage type, access modes, and capabilities
- Glossary pages for Kubernetes storage concepts

## Development
```bash
npm install
npm run dev
```

## SEO focus
Primary keywords: "CSI drivers" and "CSI driver list".
